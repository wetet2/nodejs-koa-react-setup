# nodejs-koa-react-setup
This is base setup with babel 7 and webpack 4 for starting React development

**You have to install and build modules before starting server**
```
$ yarn
```
```
$ yarn build:server
```
```
$ yarn build:client
```

**after that,**
```
$ yarn server:start
```

you can see the website below

http://localhost:4000/
